Smart-Ambient-Node: 基于 ESP8266 的私有云环境监测系统这是一个完整的物联网解决方案，实现了从物理环境感知到私有化云端数据可视化的全链路打通。项目采用 ESP-12F 作为核心控制器，通过 MQTT 协议将 DS18B20 采集的温度数据实时上报至本地 Docker 部署的 ThingsBoard 平台。🛠️ 技术栈 (Tech Stack)维度技术选型硬件端ESP8266 (ESP-12F), DS18B20 温度传感器, 0.96" OLED (SSD1306)嵌入式开发C++/Arduino, PlatformIO, U8g2, PubSubClient传输协议MQTT (Message Queuing Telemetry Transport)后端/云端ThingsBoard (Professional IoT Platform), Docker调试工具MQTTX, Serial Monitor📐 系统架构 (Architecture)感知层：DS18B20 高精度采集环境温度。边缘层：ESP8266 处理原始信号，驱动 OLED 本地实时显示，并通过 WiFi 连接局域网。传输层：遵循 MQTT 标准协议，以 JSON 格式封装遥测数据。平台层：基于 Docker 的 ThingsBoard 私有化部署，负责数据解析、存储及看板生成。✨ 核心特性 (Features)本地双监视：OLED 屏幕实时刷新，支持高帧率 UI 渲染。私有化云端：通过 Docker 一键部署，数据完全本地化存储，零延迟响应。鲁棒性设计：实现了针对无符号整数（unsigned long）比较的定时器逻辑，有效避免 millis() 溢出导致的死机。可视化看板：ThingsBoard 实时曲线与阈值颜色预警指示灯。📸 项目展示 (Demo)此处建议插入两张照片/截图：[实物运行图]：展示你的 ESP8266 和亮起的 OLED 屏幕。[云端看板图]：展示你在 ThingsBoard 里做出的温度折线图和变色灯。🚀 快速启动 (Quick Start)1. 后端部署 (Docker)Bashdocker run -it -p 9090:9090 -p 1883:1883 --name my-tb -d thingsboard/tb-postgres
访问 localhost:9090 进行初始配置。2. 硬件烧录 (PlatformIO)修改 src/main.cpp 中的 WiFi SSID、Password 以及最新的主机 IP 地址，然后运行：Bashpio run --target upload
